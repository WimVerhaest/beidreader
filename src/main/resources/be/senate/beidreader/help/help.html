Using the BeIDReader

Purpose:

This software reads EID-data from the EID-card and allows to add the data into a csv-file. This csv-file can be imported into the PACXS-database by the use of the PACXS-software.

Prerequisites

  • A PC to which a (valid) Eid-cardreader is connected (and tested with the EIDViewer);
  • A Java Run-Time version 1.8.x
  • The given jar-file: BeIDReader-jar-with-dependencies.jar

Typical use-case

A (potential) cardholder presents himself with his EID-card to the officer. The officer reads the data of Eid-card, using BeIDReader, and adds it to the csv-file.

From time to time the officer loads the csv-file into the PACXS-database using the PACXS-software.

Use:

The program can be started with the command:

java -jar BeIDReader-jar-with-dependencies.jar <csv-filename.csv>

If the file exists, the contents is read and the file opened. Otherwise, we start with a fresh situation (new-file-to-be-saved).

To read the data of an Eid-card, just enter the card into the reader and wait a little...

As soon as the data appears on the screen it can be added to the existing list by pushing the button 'Add'.

Entries can be deleted from this list as well.

Of course, the file has to be 'saved' in order to keep the newly added data.

Remark: The (BeIDReader)-software does not (yet?) check wether data of the same person is already present in the csv-file. It just adds the new information.
